# Bell Customer Churn Prediction
Predictive modeling project for Bell Canada to identify high-risk customers and support retention strategies.
Project Overview

Business problem: Bell wants to reduce churn and increase campaign ROI.

Dataset: Building-level TV subscriber and campaign data (2016–2018).

Goal: Predict post-campaign churn and identify actionable segments.

Key Features

1. Data cleaning and preprocessing

2. Feature engineering (e.g., revenue per customer, competitor influence, building penetration)

3. Machine learning models: Logistic Regression, Random Forest, XGBoost

Model evaluation: ROC-AUC, F1-score, precision-recall

Explainability: SHAP / feature importance

Dashboard for visualizing churn risk and ROI

Usage

Run notebooks in order: data cleaning → feature engineering → modeling → evaluation.

Open dashboards/ folder to view Power BI dashboards.
